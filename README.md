# Predictive Maintenance for Electric Vehicles (EVs) ⚡🚛
**MBA Capstone Project | University Canada West**

## 📌 Project Overview
This project explores the transition from reactive to predictive maintenance within the Canadian Electric Vehicle (EV) fleet landscape. By leveraging machine learning, this analysis identifies patterns in battery degradation and motor performance to forecast maintenance needs before failures occur.

## 📈 Business Value
- **Cost Reduction:** Minimizing emergency repairs and extending asset lifecycle.
- **Operational Uptime:** Reducing unscheduled downtime for commercial fleets.
- **Sustainability:** Optimizing energy consumption and battery health management.

## 🛠️ Technical Implementation
- **Language:** Python
- **Libraries:** Pandas (Data Wrangling), Scikit-Learn (Modeling), Matplotlib/Seaborn (Visualization)
- **Algorithms Used:** Random Forest Regressor / Logistic Regression (for failure prediction)

## 📂 Repository Structure
- `data/`: Contains anonymized/sample EV telemetry data.
- `notebooks/`: Jupyter Notebook containing the end-to-end analysis (EDA, Preprocessing, Modeling).
- `models/`: Saved model weights or serialized objects.
